# SBF converter golden fixtures

`units.json` holds golden SBF unit-conversion fixtures generated by **MegaMek's own**
`SBFUnitConverter` — the independent oracle for `crates/core/src/engine/sbf.rs` (`convert_unit`).
Each fixture records the Alpha Strike element **inputs** (so the Rust test feeds identical inputs to
neurohelmet's parser + converter) and the SBFUnit **output** MegaMek produced. Consumed by the
`golden_vs_megamek_converter` test.

## Regenerate

Requires a local MegaMek clone at `~/dev/MegaMek`, built once (`./gradlew megamek:jar` — the
`build/launch4j/lib` runtime jars must exist). Then:

```sh
cd ~/dev/MegaMek
JAR=megamek/build/libs/MegaMek.jar
javac -cp "$JAR" -d /tmp/sbfgolden /path/to/neurohelmet/data/sbf-goldens/SbfGolden.java
java -cp "megamek/build/launch4j/lib/*:/tmp/sbfgolden" SbfGolden 2>/dev/null \
  | awk 'f||/^\[/{f=1;print}' > /path/to/neurohelmet/data/sbf-goldens/units.json
```

The `awk` strips a stray log4j reconfiguration line that precedes the JSON. Edit the `FIXTURES`
table in `SbfGolden.java` to add units (paths are relative to `data/mekfiles/` in the clone).
