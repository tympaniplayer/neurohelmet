# Alpha Strike

The fast-play card game, four cards to a screen. Create an Alpha Strike session with **`A`**.

## What it tracks

- **Armor & structure** — pip tracks for each.
- **Damage by range band** — short / medium / long values.
- **Heat** — a 0–4 heat dial, including overheat.
- **Criticals** — the four Alpha Strike critical types.
- **Special abilities** — the card's ability line, kept visible while you play.

Alpha Strike is costed in **PV** (Point Value), and the roster total tracks against any
[point budget](../guides/force-generation.md) you set.

## Ground scale

Alpha Strike has an optional **1:1 ground (hex) scale** for playing elements on a hex map rather than
in the abstract. Press **`?`** for the Alpha Strike keymap.
