# Game systems

Neurohelmet tracks six BattleTech systems. Each session is **locked to one system** at creation, so pick
the one that matches the game you're playing. Costing follows the system: Classic and Override use
**BV** (Battle Value); the Alpha Strike family uses **PV** (Point Value).

| System | Scale | Create | Cost |
|--------|-------|:------:|:----:|
| **[Classic](classic.md)** (Total Warfare) | one 'Mech / vehicle / squad | `n` | BV |
| **[Alpha Strike](alpha-strike.md)** | one element | `A` | PV |
| **[Override](override.md)** | one unit | `O` | BV |
| **[Strategic BattleForce](strategic-battleforce.md)** (IO:BF) | formations | `B` | PV |
| **[BattleForce](battleforce.md)** (IO:BF) | a company of lances | `F` | PV |
| **[Abstract Combat System](abstract-combat-system.md)** (IO:BF) | planetary invasion | `C` | PV |

A few things that apply everywhere:

- **Press `?` in any mode** for its exact keybindings — they differ from system to system.
- From a **Classic** or **Alpha Strike** session you can peek at any unit's **Override** card on the
  fly with `O`, without starting a separate Override session.
- The roster, [named sessions](../guides/sessions.md), autosave, undo, and a
  [force point budget](../guides/force-generation.md) are shared across all six.
