# BattleForce

Standard **BattleForce** from *Interstellar Operations: BattleForce* — per-element Alpha Strike
tracking at hex scale, organized into lances. Create a BattleForce session with **`F`**.

Where [Strategic BattleForce](strategic-battleforce.md) abstracts elements up into formation-scale
Units, standard BattleForce keeps each **element** tracked individually (as an Alpha Strike card) and
groups them into **lance-Units** for command and morale.

## What it tracks

- **Per-element AS cards** — each element retains its own armor, structure, and damage.
- **Hex scale** — positions and movement at the BattleForce hex level.
- **Criticals** — the BattleForce critical-hit table.
- **Morale** — tracked manually.

BattleForce is costed in **PV**. Press **`?`** for the BattleForce keymap.
