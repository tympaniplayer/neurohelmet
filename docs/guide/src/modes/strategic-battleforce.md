# Strategic BattleForce

The **Strategic BattleForce** (SBF) system from *Interstellar Operations: BattleForce* — Alpha Strike
elements grouped up and tracked at formation scale. Create an SBF session with **`B`**.

## Structure

Elements are organized into a two-level hierarchy:

- **Units** — 1–6 Alpha Strike elements combined into one SBF Unit.
- **Formations** — Units grouped into a Formation.

## What it tracks

- **Unit-scale stats** — the combined armor, damage, and movement of each SBF Unit.
- **Spillover damage** — damage that carries past a Unit's capacity.
- **Criticals** — the SBF critical-hit table.
- **To-hit** — the attack-modifier breakdown at this scale.
- **Morale** — tracked manually, the way the tabletop plays it.

SBF is costed in **PV**. Press **`?`** for the SBF keymap.
